## 注意要点

- 使用好缩进
  - 缩进是python的灵魂，不同的缩进格式可能会导致程序语法错误
- 控制语句（类似于if，else）后面一定要有冒号：
- python里面，变量没有类型，不需要声明
- BIF（Built-in-function）内置函数
- 逻辑与使用```and```
- ```3<4<5```在python中合法，这个被解释为```3<4 and 4<5```
- 使用```elif```代替else if
- 三元操作符：```small = x if x<y else y```
  - x<y 时返回x，否则返回y
- 对于相加操作，一定是要同一个类型的变量才能相加连接

# 相关操作

- 如果想要默认去尾（去掉小数点后面的数），即类似于java的整数除法，使用```//```

- 使用```**```表示幂运算

## 列表

普通列表

- ```member = ['a','b']```

混合列表

- ```mix = [1,'a',3.14,[1,2,3]]```

空列表

- empty=[]

添加元素

- append（），只能添加一个元素
- extend（），使用列表添加列表，可以添加多个元素
  - `member.extend(['a','b'])`
- insert（），插入元素

显示长度

- ```len(member)```

删除元素

## 字符串

### 方法

```capitalize()```将第一个字符改成大写

```casefold()```将所有字符改成小写

#### 字符串格式化

```format()```将文件转换成统一的标准

位置参数

- eg：```"{0} love {1},{2}".format("I","FishC","com")```

结果为"I love FishC.com"

关键字参数

等效于```"{a} love {b},{c}".format(a="I",b="FishC",c="com")```

- eg:```"{0:.1f}{1}".format(27.658,"GB")```

结果为"27.7GB"

这个同C语言中的```printf()```

冒号指的是格式化符号的开始

##### 相关符号

%c 格式化字符及其ASCII码

- eg:```"%c %c %c" % (97,98,99)```

结果为"a b c"

这里中间的%代表格式化的意思

应用：

`print("%d 的结果是: %d" % (1, 2))`类似于C中的printf

### 序列

```list()```将一个字符串/元组变成列表

```tuple()```将一个东西变成元组

```max()```，```min()```，```len()```，```sum()```

使用```max()```，```min()```和```sum()```数据类型必须相同

```sum()```不能用于字符串类型，必须要是一个集合

```sorted()```返回的是列表

```reversed()```，```enumerate()```和```zip(a,b)```返回的是迭代器，需要使用```list()```输出正确格式

```enumerate()```是将列表中的各个数字填上序号变成一个小的元组，多个小的元组组成了一个列表
